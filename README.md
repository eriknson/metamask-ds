# MetaMask Design System

A minimal, mobile-first design system dashboard showcasing essential UI primitives and MetaMask-specific components.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Radix UI primitives

## Components

**Primitives:** Button, Input, Card, Badge, Avatar, Switch, Skeleton, Dialog, Select, Banner

**MetaMask:** Account Picker, Token Cell, Token List, Footer Nav, Sorting, Pay With, Empty State

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the component library.
